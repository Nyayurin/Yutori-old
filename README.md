## 介绍

Yutori 是一个基于 [Satori](https://satori.chat) 协议的多平台机器人开发,
框架本体使用 Kotlin 开发,
提供 [Kotlin](https://kotlinlang.org), [Java](https://www.oracle.com/cn/java) 等 JVM 平台语言的开发支持
